Apa itu monas ?

monas adalah sebuah aplikasi dekstop/website dimana pengguna dapat mengelola data, monas juga memiliki fitur fitur yang gampang digunakan oleh user seperti tambah, edit, hapus dan print data monas juga membentuk dasar fungsionalitas untuk berinteraksi dengan data dalam sistem perangkat lunak, dan monas juga digunakan bersama dengan basis data untuk menyimpan dan mengelola informasi.

Fitur Monas

TAMBAH DATA

Pengguna dapat membuat data baru dalam aplikasi CRUD. Ini melibatkan input data baru ke dalam sistem, yang kemudian disimpan dalam basis data.

EDIT DATA

Pengguna dapat memperbarui data yang sudah ada. Ini bisa termasuk mengubah nilai-nilai tertentu dari entitas data yang ada atau mengganti data dengan data

HAPUS DATA

Pengguna dapat menghapus data yang sudah ada dari aplikasi. Ini menghapus data dari sistem atau basis data.

PRINT DATA

untuk mencetak atau membuat salinan fisik dari data yang terdapat dalam aplikasi CRUD.
